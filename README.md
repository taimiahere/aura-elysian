# SaaS Application

This is a full-stack SaaS application built with Next.js, TypeScript, Tailwind CSS, Prisma, and PostgreSQL.

## Features
- Authentication (Google + Email)
- Dashboard
- AI Assistant
- CRM & Projects
- Billing system
- Admin panel

## Tech Stack
Next.js 15, React 19, TypeScript, Prisma, PostgreSQL, Stripe, OpenAI

## Status
Production-ready project (in development / deployment stage)
